# fmjpeg2koj
JPEG2000 codec for DCMTK based on openjpeg

## Download
Source https://github.com/DraconPern/fmjpeg2koj

## Requirements
- CMake http://www.cmake.org/download/

## Third party dependency
- DCMTK http://dicom.offis.de/ 
- openjpeg http://www.openjpeg.org/
